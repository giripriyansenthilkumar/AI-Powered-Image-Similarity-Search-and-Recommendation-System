# Manual Asset Instructions

Due to editor limitations, please add the following files manually:

1. frontend/assets/logo.png
   - 64x64 or 128x128 PNG
   - Example: a simple indigo/purple circle with a white hanger or shirt icon

2. frontend/assets/placeholder.jpg
   - 128x128 or 256x256 JPG
   - Example: a neutral gray background with a faded clothing outline

3. frontend/assets/icons/upload.svg
   - Use this SVG content:
<svg width="36" height="36" viewBox="0 0 36 36" fill="none" xmlns="http://www.w3.org/2000/svg">
  <rect width="36" height="36" rx="8" fill="#e9ecef"/>
  <path d="M18 25V11M18 11L12 17M18 11L24 17" stroke="#5f3dc4" stroke-width="2.2" stroke-linecap="round" stroke-linejoin="round"/>
</svg>

Once these files are added, the frontend will display correctly.
